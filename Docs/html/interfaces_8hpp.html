Descrição detalhada

Declaração de classes que representam as interfaces do sistema.

Este arquivo contém as interfaces das camadas de apresentação e serviço do sistema. As interfaces definem os contratos que devem ser implementados pelas classes concretas, garantindo a separação de responsabilidades e a modularidade do código.

As interfaces são divididas em:

  • Camada de Apresentação: Responsável pela interação com o usuário.
  • Camada de Serviço: Responsável pela lógica de negócio e comunicação com a camada de domínio.
